# IPO Lector – Prototipo Reto 3

Prototipo interactivo de una aplicación móvil de lectura y descubrimiento de libros y audiolibros.

## Pantallas incluidas

- Inicio
- Descubrir libros
- Resultados
- Ficha del libro
- Modo lectura
- Audiolibro
- Biblioteca
- Progreso

## Funcionalidades principales

- Recomendaciones explicadas.
- Filtros por género, duración, idioma, dificultad y formato.
- Lectura accesible con modo noche, marcador y progreso.
- Audiolibro con controles grandes.
- Continuidad con “Seguir donde lo dejé”.
- Modo offline preparado.
- Métricas de progreso no intrusivas.

## Cómo subirlo a GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos al repositorio:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `.nojekyll`
3. Entra en **Settings > Pages**.
4. En **Build and deployment**, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Guarda los cambios.
6. Espera unos minutos y copia el enlace público de GitHub Pages.

## Enlace para entregar

Cuando GitHub Pages esté activado, el enlace tendrá un formato parecido a:

`https://TU_USUARIO.github.io/NOMBRE_DEL_REPOSITORIO/`
